# Nabil Ould Hamou - Portfolio

To visit my website [click here](https://nabiloh.com/)!

A minimalist portfolio website built with [Nuxt 4](https://nuxt.com/).

## Running locally

Start by cloning this repo:

```bash
git clone https://github.com/nabilouldhamou/portfolio
```

Then install the dependencies using pnpm:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```
